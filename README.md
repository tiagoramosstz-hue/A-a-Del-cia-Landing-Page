# 🍇 Açaí Delícia — Landing Page






 ## Landing page moderna e responsiva desenvolvida para apresentar produtos de açaí, com foco em experiência visual, animação imersiva e conversão de clientes.

🚀 Demonstração

👉 Projeto focado em:

Performance leve

Design moderno estilo comercial

Animação fluida estilo vídeo

🛠️ Tecnologias Utilizadas
🔹 Front-end

HTML5

CSS3 (Flexbox + Grid + Variáveis CSS)

JavaScript Vanilla (ES6)

🔹 Recursos Visuais

Animação por sequência de imagens (frame animation)

Layout responsivo

Efeitos de hover e transições suaves

Overlay com blur (glassmorphism leve)

⚙️ Funcionalidades
🎬 Hero com Animação Full Screen

Background animado ocupando 100% da tela

Simulação de vídeo usando frames (.jpg)

Atualização automática a cada 100ms

🧭 Navegação Fixa (Navbar)

Menu fixo no topo

Links com scroll para seções

Efeito hover interativo

🍧 Cardápio Dinâmico

Produtos com:

Imagem

Nome

Descrição

Preço

Layout em grid responsivo

💸 Seção de Ofertas

Destaque visual para promoções

Cards com foco em conversão

📩 Formulário de Contato

Inputs validados (required)

Estrutura pronta para integração com backend

📱 Responsividade

Adaptado para:

Desktop

Tablet

Mobile

Media queries otimizadas

🎨 Design e UX

Dark theme (roxo + verde neon)
